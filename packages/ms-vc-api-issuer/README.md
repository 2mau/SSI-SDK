# ms-vc-api-verifier

A Veramo plugin to issue credentials using the Microsoft Azure Active Directory. 

### Installation

```shell
yarn add @sphereon/ssi-sdk-ms-vc-api-issuer
```

### Build

```shell
yarn build
```
